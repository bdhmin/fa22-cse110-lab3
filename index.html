Q: What does the annotate feature do?

A: The annotate feature allows users to highlight certain texts and save them as annotations.

More about highlighting

The highlight tool doesn't manipulate the DOM anymore. Instead, it creates a box behind the text that is still hoverable by using the pixel coordinates of the cursor.

The highlight tool also is built to be overlappable without excessive computations.

The highlighter can now be used with more versatility, allowing the platform to become a more powerful tool for learning
